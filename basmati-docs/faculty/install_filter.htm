Installing the Basmati Filter

You will need to install a "filter" file so that Grade Machine can properly export your grades to Basmati. This is a simple process and only need to be performed once for each computer that you use. Follow the steps below:
  1. Click here to download the filter file (called bas_os.flt).
  2. Your web-browser should prompt you to save this file. If given the choice, save this file to the FILTERS folder within your Grade Machine folder.
  3. If you weren't given a choice where you should save the file, locate the file and then copy it to the FILTERS folder within your Grade Machine folder.
This filter will export all of your assignments for all grading periods within your Grade Machine class. Other filters are available that will allow you to specify which assignments and grading periods should be exported. Here is the complete list of filters:
  • bas_os.flt -- This is the original filter which will export all assignments from all grading periods. In Grade Machine's filter list, this is seen as Basmati (OS).
  • bas_osa.flt -- This filter will ask you which assignments you would like to export. In Grade Machine's filter list, this is seen as Basmati (OS-Assign Ask).
  • bas_impt.flt -- This is an IMPORT filter which can be used in conjunction with Basmati's "Get Passwords" page. If you save the contents of the "importable" passwords page to a text file, you can use this filter to import passwords back into Grade Machine. This filter will import into the MISC 2. column of the student window in Grade Machine.
  • bas_65os.flt -- This filter can be used if you are using Grade Machine version 6.5 or later.
